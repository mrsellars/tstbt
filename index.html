The template a.k.a blog engine for developers

tstbt is for you if:

  • you know some HTML
  • you don't need dynamic content shizzle
  • you would like to blog, but still procrastinating because you are searching for the
    right publishing tool
  • you love static pages and blazing fast websites
  • you like to have a GIT based workflow
  • you would like to deploy the blog everywhere. For example on your Raspberry Pi,
    S3, GitHub Pages and so on

How to blog?

Just checkout the template.html (https://github.com/mrsellars/tstbt) and adjust it.

Have fun!
